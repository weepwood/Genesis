# Genesis

## 文明进化模拟游戏

一个基于 Godot 的文明演化沙盒游戏。

核心理念：

> 玩家创造环境与规则，而不是直接控制文明。观察人类如何在随机事件、环境压力和科技演化中形成不同未来。

## 当前版本

MVP v0.1：10分钟文明进化实验

目标：

- 世界生成
- 人类部落模拟
- 自动资源循环
- 科技随机发现
- 文明属性演化
- 历史事件记录

## 设计方向

融合：

- WorldBox 的世界模拟
- The Sapling 的进化思想
- 文明系列的科技演化
- RimWorld 的故事生成

## 开发

引擎：Godot 4.x
语言：GDScript
