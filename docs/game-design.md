# Genesis 游戏设计方案

## 1. 文明自动演化算法

文明状态：

- population 人口
- food 食物
- intelligence 智慧
- cooperation 合作
- aggression 侵略性
- curiosity 探索欲
- technology 科技水平

每个时间周期：

1. 消耗资源
2. 人口增长或减少
3. 根据环境压力生成需求
4. 根据文明属性尝试发现科技
5. 触发随机历史事件
6. 更新文明特征

## 2. 科技生成系统

科技不是固定树，而由以下因素生成：

需求 + 环境 + 文明特征 + 历史事件

例如：

寒冷环境 + 生存压力 -> 保暖技术

稀有矿物 + 好奇心 -> 冶炼技术

战争压力 + 侵略性 -> 武器技术

## 3. 文明 AI

采用简单 Utility AI：

文明根据收益评估行为：

- 扩张
- 研究
- 战争
- 贸易
- 迁徙

## 4. 第一版玩法

10分钟模拟：

玩家创建世界。

投放人类种子。

每分钟选择一次影响：

- 资源
- 环境
- 文化倾向

最终生成文明历史报告。
